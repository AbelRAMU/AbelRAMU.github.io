Obtener el listado de todos los alumnos de la facultadConsulta 1

SELECT * FROM alumnos

Obtener el listado de cursos (nombres) y las asignaturas que tieneConsulta 2

SELECT curso.nombreDescriptivo, asignatura.nombre FROM curso INNER JOIN asignatura ON curso.idCurso=asignatura.curso

Obtener el listado de alumnos que tengan becaConsulta 3

SELECT alumno.nombre, alumno.apellido1, alumno.apellido2 FROM alumno WHERE alumno.beca LIKE 'si%'

Obtener el listado de profesores (nombre y apellido), y su numero de contactoConsulta 4

SELECT profesor.nombre,profesor.apellido1,profesor.apellido2, tlfcontactoprof.telefono FROM profesor INNER JOIN tlfcontactoprof ON tlfcontactoprof.idProfesor=profesor.idProfesor

Obtener el listado de los alumnos (todos los campos) que estan inscritos a la asignatura de programacionConsulta 5

SELECT alumno.* FROM alumno INNER JOIN matricula ON alumno.idAlumno = matricula.idAlumno INNER JOIN asignatura ON matricula.idAsignatura = asignatura.idAsignatura WHERE asignatura.nombre = "Programacion"

Obtener listado de los profesores y asignaturas que impartenConsulta 6

SELECT profesor.*, asignatura.* FROM profesor INNER JOIN asignatura ON profesor.idProfesor = asignatura.coordinador ORDER BY profesor.nombre, asignatura.nombre

Mostrar la asignatura que tiene mas alumnos inscritos y su totalConsulta 7

SELECT asignatura.nombre AS nombre_asignatura, COUNT(matricula.idAlumno) AS total_alumnos_inscritos FROM asignatura INNER JOIN matricula ON asignatura.idAsignatura = matricula.idAsignatura GROUP BY asignatura.idAsignatura ORDER BY total_alumnos_inscritos DESC

Obtener el listado de alumnos que les da clase con el profesor id PR048 (jose manuel), sin importar en que asignatura este inscritoConsulta 8

SELECT alumno.* FROM alumno INNER JOIN matricula ON alumno.idAlumno = matricula.idAlumno INNER JOIN asignatura ON matricula.idAsignatura = asignatura.idAsignatura WHERE asignatura.coordinador = 'PR016'

Obtener lista de cursos con asignatura y maestro que las imparteConsulta 9

SELECT curso.nombreDescriptivo, asignatura.nombre, profesor.nombre, profesor.apellido1, profesor.apellido2 FROM curso JOIN asignatura ON curso.idCurso = asignatura.curso JOIN impartir ON asignatura.idAsignatura = impartir.idAsignatura JOIN profesor ON impartir.idProfesor = profesor.idProfesor ORDER BY curso.idCurso, asignatura.idAsignatura, profesor.idProfesor;

Obtener listado de alumnos con asignaturas que cursan, profesor que las imparte y curso a que pertenecenConsulta 10

SELECT alumno.nombre, alumno.apellido1, alumno.apellido2, asignatura.nombre, profesor.nombre, profesor.apellido1, profesor.apellido2, curso.nombreDescriptivo FROM alumno JOIN matricula ON alumno.idAlumno = matricula.idAlumno JOIN asignatura ON matricula.idAsignatura = asignatura.idAsignatura JOIN impartir ON asignatura.idAsignatura = impartir.idAsignatura JOIN profesor ON impartir.idProfesor = profesor.idProfesor JOIN curso ON asignatura.curso = curso.idCurso ORDER BY alumno.idAlumno, asignatura.idAsignatura;
